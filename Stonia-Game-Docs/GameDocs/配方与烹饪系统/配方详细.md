# 配方详细

本文档记录 Stonia 中制造与铸造的配方规则。数值与材料组合以当前 `Stonia.html` Web 实现为准。

> 烹饪相关配方见 [[烹饪详细]]。

## 1. 概述

合成界面使用两个材料格加一个结果格。玩家可以单击背包材料后放入材料格，也可以拖拽材料。

配方分为两类：

- **制造**：靠近破旧工具箱时使用。
- **铸造**：靠近铸造台时使用。

## 2. 通用规则

- 玩家须站在对应设施所在格，或上下左右相邻格。
- 两个材料格均须放入有效材料，才能产出结果（单材料配方仅存在于铸造台的金银块兑换）。
- 材料键名按物品 id 字母序用 `+` 连接（如 `iron+stone`）。
- 无效组合会提示「配方不正确」。
