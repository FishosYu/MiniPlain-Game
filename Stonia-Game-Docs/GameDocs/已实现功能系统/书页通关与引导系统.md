# 书页通关与引导系统

本文档记录当前 `Stonia.html` 中遗落书页、水晶球通关、冒险手册、物品图鉴、开发者工具和屏幕切换等辅助系统。

## 1. 遗落书页

当前实现共有 7 张遗落书页，正文数据来自 `data/pages.js` 的 `window.MINIPLAIN_PAGES`。

发现方式：

- 凿石时小概率发现书页。
- 书页进度落后时，保底逻辑会强制补出下一张未收集书页。
- 地图剩余石头不足以容纳缺失书页时，也会触发兜底。

发现效果：

- 加入 `collectedPages`。
- 记录图鉴物品 `page`。
- 获得等于当前等级的经验。
- 解锁冒险手册中的“第一张书页”成就。
- 可在背包中阅读。

主要实现：

- `PAGE_NUMS`
- `PAGES`
- `PAGE_TOTAL`
- `collectedPages`
- `pageCount`
- `applyPageGuarantee(hidden)`
- `showPage(pi)`

## 2. 水晶球

水晶球是书页提交和胜利触发入口。

出现规则概述：

- 真实凿石次数超过 70 后开始有概率出现。
- 后期概率会随凿石次数增加而提高。
- 第 121 次仍未出现时强制出现。
- 地图挖空风险较高时也会兜底出现。

交互：

- 点击水晶球打开提交界面。
- 界面显示已收集和已提交书页进度。
- 未提交书页逐张显示提交按钮。
- 已提交书页可以重新阅读。
- 全部提交后，底部出现通关按钮。

主要实现：

- `crystalBallPos`
- `shouldRevealCrystalBall()`
- `openCrystalBallUI()`
- `submittedPages`
- `isCrystalLit(x, y)`

## 3. 通关流程

完整通关链路：

1. 持续凿石探索地图。
2. 收集 7 张遗落书页。
3. 找到水晶球。
4. 在水晶球界面提交所有书页。
5. 点击水晶球界面中的完成按钮。
6. 显示胜利弹窗。

胜利弹窗提供：

- 再来一局。
- 返回地图继续查看。

重新开始会重置：

- 玩家体力、经验、等级和位置。
- 地图、背包、箱子、合成栏。
- 书页、水晶球、商店购买记录。
- 冒险手册和开发者工具状态。

主要实现：

- `victory-modal`
- `btn-crystal-win`
- `restartGame()`
- `openNewGameConfirm()`

## 4. 冒险手册

冒险手册是当前最明确的玩家引导系统，入口是状态栏书本按钮。

当前共有 5 章：

| 章节 | 主题 | 成就 |
|------|------|------|
| 第一章：石缝醒来 | 开局探索和工具箱 | 找到工具箱、凿开石头、获得第一把工具 |
| 第二章：火与锅 | 篝火、铁锅和第一份料理 | 点起篝火、获得铁锅、做出第一份料理 |
| 第三章：田与面粉 | 小麦和面粉链条 | 种下小麦、磨出面粉、烤出面包 |
| 第四章：秘密菜单 | 铁板和汉堡链条 | 获得铁板、做出吮指原味鸡、做出鸡肉汉堡 |
| 第五章：遗落书页 | 书页和水晶球 | 发现书页、发现水晶球、提交书页 |

解锁规则：

- 第一章默认可看。
- 后续章节需要前一章全部成就完成。
- 新章节可读但未查看时，书本按钮会显示新页提示。
- 开发者模式可临时切换为全部完成视图。

主要实现：

- `ADVENTURE_PAGES`
- `ADVENTURE_ACHIEVEMENT_MAP`
- `adventureAchievements`
- `adventurePageComplete(index)`
- `canOpenAdventurePage(index)`
- `unlockAdventureAchievement(id)`
- `renderAdventureHandbook()`
- `openAdventureHandbook()`

## 6. 物品图鉴

图鉴用于记录玩家已经见过的物品和说明。

入口：

- 主菜单图鉴按钮。
- 游戏内图鉴按钮。

规则：

- 只显示已发现物品的真实名称、图标、分类和说明。
- 未发现物品显示为 `?` 或未发现。
- 发现记录写入浏览器 `localStorage`。
- 开发者模式可切换显示全部图鉴。

数据来源：

- `data/items.js` 的物品分类和文案。
- `data/i18n.js` 与 `data/locales/*.js` 的本地化文本。

主要实现：

- `DISCOVERY_STORAGE_KEY`
- `discoveredItems`
- `loadDiscoveredItems()`
- `saveDiscoveredItems()`
- `discoverItem(id)`
- `codexEntry(id)`
- `renderCodex()`

## 7. 主菜单与屏幕切换

当前有三个主要屏幕：

- 主菜单。
- 物品图鉴。
- 游戏界面。

切换时会同步处理：

- 显示/隐藏对应 screen。
- 进入游戏时尝试播放 BGM。
- 离开游戏时暂停 BGM。
- 移动端竖屏与方向控制状态。
- 打开图鉴时重新渲染图鉴内容。

主要实现：

- `showScreen(screen)`
- `playGameBgm()`
- `pauseGameBgm()`
- `updateDeviceMode()`

## 8. 开发者工具

开发者工具用于调试玩法和文档核对。

入口：

- 按反引号 `` ` ``。
- 输入口令 `fishcanfly`。

当前功能：

- 忽略体力限制。
- 忽略背包容量限制。
- 忽略移动限制。
- 关闭昼夜限制。
- 快速挖矿。
- 一键清除所有石头。
- 全工具判定。
- 显示全部图鉴。
- 冒险手册完成视图。
- 搜索并添加物品。

关闭开发者工具会重置大多数开发者开关，并重新渲染地图、背包和图鉴。

主要实现：

- `devToolsActive`
- `toggleDevTools()`
- `renderDevTools()`
- `closeDevTools()`
- `clearAllStonesDev()`
- `devIgnoreStaminaLimit`
- `devIgnoreInventoryLimit`
- `devIgnoreMovementLimit`
- `devFastMine`
- `devShowAllCodex`
- `devAdventureComplete`
