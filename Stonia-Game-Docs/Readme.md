# Stonia 文档工作空间

> 这是 **Stonia / 小石之原** 项目的文档工作空间（Documentation Workspace），推荐使用 **Obsidian + Claude Code** 打开和管理。

## 推荐工具

- **Obsidian** — 用于浏览、编辑和整理文档笔记，支持双向链接、标签、图谱等功能（[下载 Obsidian](https://obsidian.md/download)）
- **Claude Code** — 配合 Obsidian 使用，可通过 CLI 与仓库交互，辅助代码文档编写和项目管理

## 目录说明

| 目录/文件 | 用途 |
|-----------|------|
| `.obsidian/` | Obsidian 工作区配置，包含启用的核心插件、外观主题、图谱设置等 |
| `.claude/` | Claude Code 本地设置 |
| `GameDocs/` | 正式游戏文档，记录玩法流程、代码结构、资产需求与已实现系统 |
| `GameDocs/配方与烹饪系统/` | 制造、铸造、烹饪和加工配方细节 |
| `GameDocs/已实现功能系统/` | 按系统记录当前已实现的探索地图、资源设施、种植动物、商店、书页通关、冒险手册和图鉴等功能 |
| `Stand Up/` | 日常站会记录，用于跟踪进度和同步信息 |
| `代办/` | 任务与待办事项清单 |
| `想法池/` | 功能构思、设计思路、备忘等非结构化内容 |

## 当前已实现功能概括

当前 Web 版已经具备完整的轻探索循环：玩家从 11×11 石头地图中心开始，通过凿石获得资源、经验和随机发现，再用工具箱、铸造台、篝火、铁板、石磨、料理台和烤箱推进制造与料理链条。背包容量、箱子储物、设施放置、种植、小麦成长、鸡牛狼互动、钓鱼挤奶、金银盒子商店、冒险手册、物品图鉴、书页提交和水晶球通关流程均已实现。

详细拆分见 [[已实现功能总览]]。
