# 烹饪详细

本文档记录 MiniPlain 中烹饪系统的规则与配方。数值与材料组合以当前 `index.html` Demo 实现为准。

> 制造与铸造配方见 [[配方详细]]。

## 1. 概述

烹饪在篝火旁进行，是玩家将采集到的食材转化为可食用料理的主要方式。

## 2. 前置条件

玩家须同时满足以下条件才能烹饪：

1. 站在篝火所在格，或上下左右相邻格。
2. 背包中持有铁锅（铸造台：铁块 + 铁块）。

不满足条件时，放入食材会提示「烹饪需靠近🔥篝火（四格内）」或「烹饪需在篝火旁，且背包中有铁锅🍳」。

## 3. 界面与交互

- 合成面板在满足烹饪条件时切换为「烹饪」模式。
- 两个材料格 + 一个结果格；确认按钮文案为「烹饪」。
- 烹饪栏仅可放入食材类物品；石材、木材等制造材料须在其他设施处合成。
- 若材料格中有食材但不在篝火旁或无铁锅，会提示相应错误。

### 3.1 模式优先级

当材料格内容变化时，系统按以下逻辑决定当前合成模式：

1. 材料格含食材 → 烹饪（须满足篝火 + 铁锅条件）。
2. 材料格含石材/木材等制造材料 → 铸造或制造（须靠近对应设施）。
3. 无材料时，按 烹饪 > 铸造 > 制造 显示可用模式。

## 4. 食材分类

### 4.1 可放入烹饪栏的食材

蔬菜、土豆、胡萝卜、鸡蛋、鸡肉、牛肉、鱼、小麦。

### 4.2 蔬菜类

- 蔬菜
- 土豆
- 胡萝卜

### 4.3 肉类

- 鸡蛋
- 鸡肉
- 牛肉

### 4.4 锅具

当前仅实现铁锅；下表「使用的锅具」列预留不同锅具对应不同配方，现阶段默认均为铁锅。

| 锅具 | 获取方式 | 备注 |
|------|----------|------|
| 铁锅 | 铸造台：铁块 + 铁块 | 当前唯一可用锅具 |
